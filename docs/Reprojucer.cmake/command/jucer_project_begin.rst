jucer_project_begin
===================

Indicate the beginning of a JUCE project.

::

  jucer_project_begin(
    [JUCER_VERSION <jucer_version>]
    [PROJECT_FILE <jucer_file_path>]
    [PROJECT_ID <jucer_project_id>]
  )

You should call this command before any other :ref:`jucer_* command
<Reprojucer-commands>`.
